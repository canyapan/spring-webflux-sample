# spring-webflux-sample
A simple echo rest API as an example of how to implement a non-blocking rest APIs using Spring WebFlux

## Build
`./gradlew clean build`

## Run
`./gradlew bootRun`

## Curl
`curl http://localhost:8080/api/v1/echo/Hello%20WebFlux%20Enabled%20REST%20API`

## Package
`./gradlew bootJar`
